# ハンバーガーメニュー付き Web サイト

このプロジェクトは、HTML/CSS/JavaScript を使ったシンプルなレスポンシブメニューのサンプルです。

## 使い方

1. VS Code でこのフォルダを開く
2. `index.html` をブラウザで開く
3. コードを編集して Git でバージョン管理してみよう！

## Git練習ポイント

- `git init` して履歴をつける
- `git status`, `git diff`, `git log` の確認
- `git branch` で新しい作業ブランチを作る
- GitHub にリモートリポジトリを作って push
